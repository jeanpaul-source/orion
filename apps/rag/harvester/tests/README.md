# ORION Harvester Tests

Complete test suite for the ORION harvester application with fixtures, mocks, and comprehensive provider testing.

See full documentation in this file for setup, running tests, and writing new tests.

## Quick Start

```bash
# Install dependencies
pip install -e .[dev]

# Run all tests
pytest tests/ -v

# Run with coverage
pytest tests/ --cov=src --cov-report=html
```

## Test Coverage Goals

- Provider Factory: 90%+
- Base Provider: 85%+
- GitHub Provider: 80%+
- Overall: 60%+
